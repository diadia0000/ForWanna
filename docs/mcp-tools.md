# MCP Tools & Plugins

## 必要

- [modelcontextprotocol/servers](https://github.com/modelcontextprotocol/servers) — 官方 MCP server 集合（filesystem / git / memory / fetch）
- [microsoft/playwright-mcp](https://github.com/microsoft/playwright-mcp) — 瀏覽器自動化，測試 WebRTC Host+Client
- [AgentDeskAI/browser-tools-mcp](https://github.com/AgentDeskAI/browser-tools-mcp) — 截圖、console log 擷取

## 建議

- [zueai/mcp-manager](https://github.com/zueai/mcp-manager) — GUI 管理所有 MCP server
- [wong2/mcp-cli](https://github.com/wong2/mcp-cli) — 命令列測試 MCP server

## Codegraph（已安裝）

- [codegraph](https://github.com/jannikbertram/codegraph) — tree-sitter 知識圖，多 agent 防踩踏核心工具
